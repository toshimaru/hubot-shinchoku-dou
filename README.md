# hubot-shinchoku-dou

Yet another 進捗どうですか

## Installation

1. `npm install hubot-shinchoku-dou --save`
2. "hubot-shinchoku-dou" を external-scripts.json に追加
4. Reboot Hubot

## Commands

```
hubot 進捗
hubot shinchoku
hubot 進捗どうですか
```
